# Bosch Building Technologies - HVAC Systems Ontology

> Version 2.0.0

## Overview

HVAC system ontology to describe  HVAC entities like  `Dampers`, `Pumps`, `Thermal Distributors` (e.g. `AHU`), `Thermal Exchanger`, and `Thermal Sources` (e.g. `Chillers`).
